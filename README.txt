Veerendra Portfolio (Static Site)

Files:
- index.html
- styles.css
- script.js
- projects.js (experience data)
- veerendra.png (profile image)
- github.png (GitHub icon)
- VEERENDRA_resume.pdf (resume)

How to edit experience:
Open projects.js and update window.EXPERIENCE entries.

How to run locally:
Open index.html in a browser, or use a simple local server:
  python3 -m http.server 8080
Then visit http://localhost:8080

Deploy:
Upload the folder contents to any static host (GitHub Pages, Netlify, Vercel, S3, etc.).
